## Navbar Contract

This section is Phase 1 specific. Navbar is the primary interactive component built in this phase.

| State | Background | Blur | Border | Logo Color | Link Color |
|-------|-----------|------|--------|-----------|-----------|
| Default (scroll < 80px) | `transparent` | none | none | `#F5F5F5` | `#F5F5F5` |
| Scroll (scroll >= 80px) | `rgba(13,13,13,0.85)` | `backdrop-blur-md` (12px) | `border-b border-white/8` | `#F5F5F5` | `#F5F5F5` |
| Mobile overlay open | `#0D0D0D` (solid) | none | none | `#F5F5F5` | `#F5F5F5` |

Transition: `transition-all duration-300 ease-out` on `background-color`, `backdrop-filter`, and `border-color`.

Nav link hover: animated underline. Implementation â€” `::after` pseudo-element scales from `scaleX(0)` to `scaleX(1)` on hover, `transition-transform duration-200`. Color: `#F5F5F5`. Active state: underline persists (no scale animation reset).

Logo: Text `"CLMC"` â€” Geist, weight 700, size 20px, color `#F5F5F5`, letter-spacing `-0.02em`.

Hamburger: 3-line icon morphs to X on open. Lines are `2px` height, `24px` width, `#F5F5F5`. CSS transform animation: top line rotates `+45deg`, bottom line rotates `-45deg`, middle line fades to `opacity: 0`. Duration: `200ms ease-out`.

Mobile overlay: full-viewport `fixed inset-0 z-50 bg-[#0D0D0D]`. Nav links are Geist, 36px, weight 700, centered vertically and horizontally. Vertical stack with `40px` gap between links.

## Accessibility Contract (Phase 1)

Applies to Navbar and Footer specifically (per A11Y-01, WCAG 2.2 AA):

- Navbar `<nav>` has `aria-label="Primary navigation"`
- Mobile menu button: `aria-expanded` toggles `true`/`false`; `aria-controls` points to menu panel ID
- Mobile overlay: `role="dialog"` with `aria-modal="true"` and `aria-label="Navigation menu"`; focus trapped inside when open; `Escape` key closes menu
- All nav links: visible focus ring (`outline: 2px solid rgba(255,255,255,0.80)`)
- Active nav link: `aria-current="page"` on matching route
- Footer `<nav>` has `aria-label="Footer navigation"`
- Color contrast: `#F5F5F5` on `#0D0D0D` = 18.7:1 contrast ratio â€” exceeds WCAG AA (4.5:1) and AAA (7:1)
- `prefers-reduced-motion`: Navbar scroll transition and hamburger morph animation wrapped in `@media (prefers-reduced-motion: no-preference)` â€” static fallback renders instantly (DSGN-04, REQUIREMENTS.md)
