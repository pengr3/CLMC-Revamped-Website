**Part A: Fix nested `<main>` elements across 9 pages**

For each of these 9 files, find the `<main ...>` opening tag and `</main>` closing tag and replace with `<div ...>` and `</div>`:

1. `app/about/page.tsx`: `<main>` -> `<div>`
2. `app/about/ceo/page.tsx`: `<main className="bg-surface-primary">` -> `<div className="bg-surface-primary">`
3. `app/services/page.tsx`: `<main>` -> `<div>`
4. `app/projects/page.tsx`: `<main className="min-h-screen bg-surface-primary">` -> `<div className="min-h-screen bg-surface-primary">`
5. `app/projects/[slug]/page.tsx`: `<main className="min-h-screen bg-surface-primary">` -> `<div className="min-h-screen bg-surface-primary">`

**IMPORTANT:** Only replace `<main` and `</main>` tags. Do NOT change anything else in these files. Preserve all classNames exactly.

**Part B: Add global focus indicator CSS in `globals.css`**

Add the following rule inside the existing `@layer base` block in `globals.css`. If no `@layer base` block exists, add one after the theme blocks. This provides a safety net for any `<a>` or `<button>` that lacks inline Tailwind focus-visible classes:

```css
/* WCAG 2.2 AA — Global focus indicator safety net (D-10) */
a:focus-visible,
button:focus-visible {
  outline: 2px solid var(--color-interactive-focus);
  outline-offset: 2px;
  border-radius: var(--radius-sm);
}
```

**Part C: Verify prefers-reduced-motion (D-12)**

Check `globals.css` for existing `prefers-reduced-motion` media query. It should already handle `[data-fade-up]`, `[data-clip-reveal]`, `[data-stagger-child]` with `opacity: 1; transform: none`. Confirm this exists — do NOT modify if already correct.

If any Motion library `<motion.div>` components in Phase 3-5 pages use `initial`/`animate` props directly (not via Tailwind), they should be verified. However, do NOT add `useReducedMotion` hooks unless a specific component is found to be non-compliant — the data-attribute animations are already covered by the global CSS rule.

This matches the masonry grid layout: full width on mobile, ~half on tablet, ~third on desktop.
