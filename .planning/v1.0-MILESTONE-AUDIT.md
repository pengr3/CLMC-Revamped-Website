## Integration Gaps

### Gap 1: /about/ceo page unreachable from navigation
**Severity:** Medium
**Issue:** `app/about/ceo/page.tsx` exists with full content and metadata, but no Navbar, Footer, or About page links to it. Only discoverable via sitemap or direct URL.
**Fix:** Add a "Meet Our CEO" link from `/about` page body or add `/about/ceo` to Footer navigation.

### Gap 2: Footer missing /qms link
**Severity:** Low
**Issue:** Navbar has 7 links including `/qms`, but Footer `NAV_LINKS` only has 6 (omits QMS). Inconsistent secondary navigation.
**Fix:** Add `{ href: '/qms', label: 'QMS' }` to Footer's `NAV_LINKS` array.

### Gap 3: useClipReveal hook orphaned
**Severity:** Info (dead code)
**Issue:** `useClipReveal` exported from `components/animation/` but no component uses it or the `data-clip-reveal` attribute. Phase 4 used Motion AnimatePresence instead.
**Fix:** No action needed — hook is available for future use. Can be removed if desired.
